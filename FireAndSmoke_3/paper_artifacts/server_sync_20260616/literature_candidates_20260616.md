# Literature Candidates 2026-06-16

These are candidate references found via arXiv MCP for later verification. Do not cite them until the paper's claim and bibliographic metadata are checked.

## Sliced Inference / SAHI

1. Akyon, Altinuc, and Temizel. "Slicing Aided Hyper Inference and Fine-tuning for Small Object Detection." arXiv:2202.06934.
   - Candidate use: motivation for sliced inference and sliced fine-tuning on small objects in high-resolution imagery.
   - Reported in abstract: SAHI improved AP on VisDrone and xView across several detectors.

2. Keles et al. "Evaluation of YOLO Models with Sliced Inference for Small Object Detection." arXiv:2203.04799.
   - Candidate use: YOLO-specific sliced inference baseline for small object detection.
   - Reported in abstract: sliced inference increased AP50 across experiments on VisDrone2019Det.

3. Ahmed and Pizarro. "Maritime Small Object Detection from UAVs using Deep Learning with Altitude-Aware Dynamic Tiling." arXiv:2511.19728.
   - Candidate use: UAV high-altitude small-object tiling motivation.
   - Reported in abstract: dynamic tiling with YOLOv5 and SAHI improved mAP for small objects in maritime UAV imagery.

## P2 / Small Object Detection Heads

4. Wang and Zhao. "SOD-YOLO: Enhancing YOLO-Based Detection of Small Objects in UAV Imagery." arXiv:2507.12727.
   - Candidate use: P2 small-object detection layer precedent in UAV imagery.
   - Reported in abstract: adds a P2 layer, feature fusion, and Soft-NMS for VisDrone2019-DET.

5. Chen, Hu, and Zhang. "HierLight-YOLO: A Hierarchical and Lightweight Object Detection Network for UAV Photography." arXiv:2509.22365.
   - Candidate use: UAV small-object detection motivation and small-object head design.
   - Reported in abstract: targets small objects below 32 pixels in UAV photography.

## Fire Detection YOLO Variants

6. Pan et al. "YOLO-FireAD: Efficient Fire Detection via Attention-Guided Inverted Residual Learning and Dual-Pooling Feature Preservation." arXiv:2505.20884.
   - Candidate use: recent YOLO-based fire detection and small-fire feature preservation.

7. Pan, Wang, and Huan. "EFA-YOLO: An Efficient Feature Attention Model for Fire and Flame Detection." arXiv:2409.12635.
   - Candidate use: lightweight YOLO-based flame detection comparison.

8. Liang and Cheng. "Mirror Target YOLO: An Improved YOLOv8 Method with Indirect Vision for Heritage Buildings Fire Detection." arXiv:2411.13997.
   - Candidate use: YOLOv8 fire detection under constrained viewpoints and false-alarm concerns.
